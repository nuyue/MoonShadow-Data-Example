## Welcome!

This is a sample article for MoonShadow blog.

MoonShadow is a modern blog and developer toolkit featuring:

- 📝 Markdown writing
- 🎵 Background music player
- 🔧 26+ developer tools
- 🌐 Chinese/English bilingual support

## Getting Started

1. Copy this directory as a template
2. Modify `index.zh.md` and `index.en.md` content
3. Push to your data repository

## Multilingual Support

You can create multilingual articles:

```
articles/
└── my-post/
    ├── index.zh.md    # Chinese version
    └── index.en.md    # English version
```

Front matter also supports multiple languages:

```yaml
title:
  zh: "中文标题"
  en: "English Title"
```
