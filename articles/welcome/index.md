## 欢迎！

这是 MoonShadow 博客示例文章。

MoonShadow 是一个现代化的博客与开发者工具箱，支持：

- 📝 Markdown 写作
- 🎵 背景音乐播放
- 🔧 26+ 开发者工具
- 🌐 中英双语

## 开始写作

1. 复制这个目录作为模板
2. 修改 `index.md` 内容
3. 推送到你的数据仓库

## 多语言支持

你可以创建多语言文章：

```
articles/
└── my-post/
    ├── index.zh.md    # 中文版
    └── index.en.md    # 英文版
```

Front matter 也支持多语言：

```yaml
title:
  zh: "中文标题"
  en: "English Title"
```
